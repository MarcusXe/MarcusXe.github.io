Zsh

  • Chinese Characters shows in the form of it’s code as \245 ,Zsh is to blame.

Add 

export LC_ALL=en_US.UTF-8  
export LANG=en_US.UTF-8

to .zshrc works.

xfce4-terminal

  • Random error code on special chars.as ? or other unrecognized form.

Change it’s encoding to Unicode->UTF 8works.

Linux kernel(arch as an example)

Change kernel

  1. install first.
  2. make grub detect it.
    grub-mkconfig -o /boot/grub/grub.cfg
  3. Reboot to target kernel.
  4. Remove unnessaries.
    Done

Remove module from kernel.

1.

mkdir /etc/modprobe.d/
vim /etc/modprobe.d/blacklist.conf

add !xxxx to this file.
